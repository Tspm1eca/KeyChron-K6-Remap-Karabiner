# KeyChron-K6-Remap-Karabiner
Remap Keys with Karabiner

## Remaps
- Home -> Delete
- PgUp -> Home
- PgDn -> End
- Delete -> PgUp
- End -> PgDn

## Custom Combinations

- CapsLock + "i" -> Up
- CapsLock + "j" -> Left
- CapsLock + "k" -> Down
- CapsLock + "l" -> Right
- CapsLock + "p" -> CMD + <
- CapsLock + ";" -> CMD + >
